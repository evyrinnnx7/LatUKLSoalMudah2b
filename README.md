Program GanjilGenap2 berfungsi untuk menentukan apakah bilangan yang dimasukkan pengguna termasuk ganjil atau genap. Program ini menggunakan kelas Scanner untuk membaca input dari keyboard. Setelah pengguna memasukkan sebuah bilangan, program memeriksa hasil sisa bagi dua dari bilangan tersebut menggunakan operator %. Jika hasilnya nol, maka bilangan tersebut genap; jika tidak, bilangan tersebut ganjil. Hasilnya kemudian ditampilkan di layar.
